## 2. 状態管理設計（Zustand）

### 2.1 modelStore

**責務**: モデル構築に関する状態管理

**状態**
```typescript
interface ModelStore {
  // モデルデータ
  nodes: Node[]
  edges: Edge[]
  selectedNode: string | null
  csvMetadata: CsvMetadata | null

  // アクション
  addNode: (type: NodeType) => void
  updateNode: (id: string, updates: Partial<Node>) => void
  deleteNode: (id: string) => void
  addEdge: (edge: Edge) => void
  deleteEdge: (id: string) => void
  setSelectedNode: (id: string | null) => void
  setCsvMetadata: (metadata: CsvMetadata) => void
  getModel: () => PyMCModel
  loadModel: (model: PyMCModel) => void
  clearModel: () => void
}
```

**主要ロジック**

1. **addNode**: 新規ノードの追加
   - ユニークIDの生成
   - デフォルト位置の計算
   - nodesリストへの追加

2. **updateNode**: ノードの更新
   - 部分更新をサポート
   - 不変性を保持

3. **deleteNode**: ノードの削除
   - 関連エッジも自動削除
   - 選択状態のクリア

4. **getModel**: 現在のモデルデータを取得
   - API送信用にフォーマット

## 4. API通信層

### 4.1 apiClient

**ファイル**: `services/api.ts`

**実装**
```typescript
class ApiClient {
  private baseURL = 'http://localhost:8000/api'

  async validateModel(model: PyMCModel): Promise<ValidationResult> {
    const res = await fetch(`${this.baseURL}/validate`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(model)
    })
    return res.json()
  }

  async generateCode(model: PyMCModel): Promise<CodeGenResult> {
    const res = await fetch(`${this.baseURL}/generate-code`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(model)
    })
    return res.json()
  }

  async runInference(request: InferenceRequest): Promise<InferenceResult> {
    const res = await fetch(`${this.baseURL}/inference/run`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(request)
    })
    return res.json()
  }

  async runPriorPredictive(request: PriorPredictiveRequest): Promise<PriorPredictiveResult> {
    const res = await fetch(`${this.baseURL}/inference/prior-predictive`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(request)
    })
    return res.json()
  }

  async uploadCsv(file: File): Promise<CsvMetadata> {
    const formData = new FormData()
    formData.append('file', file)
    const res = await fetch(`${this.baseURL}/data/upload`, {
      method: 'POST',
      body: formData
    })
    return res.json()
  }

  async getDistributions(): Promise<DistributionsMap> {
    const res = await fetch(`${this.baseURL}/distributions`)
    return res.json()
  }
}

export const apiClient = new ApiClient()
```

## 5. 型定義

### 5.1 主要型

**ファイル**: `types/model.ts`

```typescript
export enum NodeType {
  VARIABLE = 'variable',
  COMPUTED = 'computed',
  DATA = 'data'
}

export interface Node {
  id: string
  type: NodeType
  name: string
  position: { x: number; y: number }
  distribution?: string
  parameters: Record<string, any>
  shape: number[] | null
  observed: boolean
  observedDataSource?: string
  expression?: string
}

export interface Edge {
  id: string
  source: string
  target: string
}

export interface PyMCModel {
  nodes: Node[]
  edges: Edge[]
  csvMetadata?: CsvMetadata
}

export interface CsvMetadata {
  file_id: string
  filename: string
  columns: ColumnInfo[]
}

export interface ColumnInfo {
  name: string
  dtype: string
  role: 'observed' | 'feature' | 'unused'
}

export interface ValidationResult {
  valid: boolean
  errors: ValidationError[]
  warnings: ValidationWarning[]
}

export interface InferenceResult {
  success: boolean
  execution_time: number
  trace_plot: string
  posterior_plot: string
  summary: Record<string, SummaryStats>
}
```
